# AES

AES (*Rijndael*) è un algoritmo di cifratura a blocchi a chiave simmetrica, adottato come standard negli Stati Uniti. Opera su matrici di byte 4×4, chiamate *State*, per blocchi da 128 bit. Aumentando la lunghezza del blocco (fino a 256 bit), cresce anche il numero di colonne della matrice, pari a (numero di bit / 32).

# AES

AES (*Rijndael*) is a symmetric-key block cipher adopted as a standard in the United States. It operates on 4×4 byte matrices, referred to as the *State*, for 128-bit data blocks. When the block size is increased (up to 256 bits), the number of columns in the State matrix increases accordingly, equal to (block size / 32).
